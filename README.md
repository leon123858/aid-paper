## What does my paper talking about?

本研究探討了將自主概念應用於數位身份管理的創新方法，提出「自主身份」（Autonomous Identity，AID）系統。該系統使每位用戶在具備道德標準的身份系統中自由的管理自己，旨在解決當前身份管理模式的固有限制。
